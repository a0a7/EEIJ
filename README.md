# Journal of Young Scientists & Engineers (JYSE)

A production-ready SvelteKit site for an open-access science and engineering journal.

## Stack

- SvelteKit (TypeScript)
- Tailwind CSS v4
- Cloudflare Pages adapter

## Local development

```bash
npm install
npm run dev
```

## Validation

```bash
npm run check
npm run lint
npm run build
```

## Build target

Configured for Cloudflare Pages via `@sveltejs/adapter-cloudflare`.
